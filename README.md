# philly-bike-finder

Indego is Philadelphiaâ€™s bike-sharing program. The Philly Bike Finder app uses Indego's public API to help users find the Indego bike stations closest to where they are currently located. Clickable map markers trigger windows that tell users how many bikes and parking docks are available at each station. 

To run this project, run `npm install` after cloning it. Then run `npm start`. This allow you to see the project running at http://localhost:3000/.
